# Janitri Dashboard Login Automation

Automation framework for the login page of the Janitri Dashboard[](https://dev-dash.janitri.in) using Java, Selenium WebDriver, TestNG, and Page Object Model (POM).

## Prerequisites
- Java 11+
- Maven 3.8+
- Chrome Browser
- ChromeDriver (compatible with your Chrome version)
